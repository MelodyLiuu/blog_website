# 2025 六角學院軟體工程師體驗營 — RWD 靜態網頁實作

##  專案概述
本專案為參與 **六角學院 30 天軟體工程師體驗營** 的實作成果。  
透過 5 場實務直播課與系統化影音課程，依據 **Figma 設計稿**完成響應式靜態網頁開發，並通過官方審核取得結業認證。

## 技術核心與執行細節
- **設計稿視覺還原**  
  使用 `HTML5 語義化標籤` 建立清晰的 DOM 結構，並透過 `CSS3 Flexbox` 還原 Figma 設計稿版面。

- **響應式網頁設計（RWD）**  
  規劃 `Media Queries` 斷點，使網站可自適應手機、平板與桌機等不同裝置，提升使用者體驗。

- **設計稿拆解與前端實作**  
  透過 Figma 解析版面與元件結構，完成前端切版並部署於GitHub Pages。
